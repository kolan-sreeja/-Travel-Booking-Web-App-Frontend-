# ✈️ Travel Booking Web App (Frontend)

A modern, responsive travel booking web application built using HTML, CSS, and Vanilla JavaScript. This project simulates a real-world flight booking experience with dynamic pricing, interactive UI, and complete booking flow — all handled on the frontend without any backend.


## 🚀 Features

### 🔍 Flight Search
- Search flights using From and To destinations
- Select travel date
- Choose number of passengers (Adults & Children)
- Filter using hotel category (3, 4, 5 star)
- Add optional services like lunch and dinner

---

### 💰 Dynamic Pricing
- Real-time price updates based on:
  - Passenger count
  - Hotel selection
  - Add-ons (lunch/dinner)
- Smooth UI updates without page reload

---

### 💳 Booking & Payment
- Book flights directly from search results
- View booking summary before payment
- Simulated payment system (frontend only)
- Booking confirmation with unique Booking ID

---

### 📦 My Bookings
- Stores all bookings using localStorage
- View booking history with:
  - Route details
  - Price
  - Date
- Cancel booking option (UI simulation)

---

### 🎧 Support / Help
- Submit travel-related queries
- Stores queries in localStorage
- View previous queries with status (Pending / Responded)

---

### 👤 User Profile
- Manage user details:
  - Name
  - Email
  - Phone
- Data persistence using localStorage

---

## 🧠 Key Highlights

- Fully dynamic frontend (no backend required)
- State-driven UI using JavaScript
- Persistent data using localStorage
- Clean and modern UI design
- Fully responsive (Mobile + Tablet + Desktop)

## 🎯 Purpose

This project demonstrates how to build a real-world travel booking interface with dynamic behavior, clean UI/UX, and frontend-only data handling without relying on backend services.

---

## ⚠️ Note

- This is a frontend-only application
- No real payment gateway is integrated
- All data is stored in the browser using localStorage
